# Dotfiles
This is my setup for bspwm
- Polybar with cappucino theme
- nitrogen
- pico
- neofetch
- fastfetch
- Betterlockscreen

# Shotcuts
>[!TIP]
> - `super + x`       # lock screen
> - `super + return`  # open kitty
> - `super + space`   # open rofi
> - `super + e`       # open dolphin
